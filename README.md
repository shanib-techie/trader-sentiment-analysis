# 📊 Trader Performance vs Market Sentiment Analysis

> Analyzing how the Crypto Fear & Greed Index influences trader behavior and profitability on Hyperliquid.

---

## 🗂️ Project Structure

```
├── analysis.ipynb          # Main analysis notebook
├── fear_greed_index.csv    # Daily Fear & Greed sentiment data

├── README.md               # This file

```

---

## 🎯 Objective

This project investigates whether **market sentiment** (as measured by the Fear & Greed Index) meaningfully affects:

- Trader **profitability** (PnL, win rate)
- Trader **behavior** (trade frequency, position size, long/short bias)
- **Trader segmentation** (high vs low leverage, frequent vs infrequent, winners vs losers)

---

## 📦 Datasets

| Dataset | Description |
|---|---|
| `fear_greed_index.csv` | Daily sentiment classification: Extreme Fear / Fear / Neutral / Greed / Extreme Greed |
| `historical_data.csv` | Hyperliquid trade-level records: Account, Timestamp IST, Side (Buy/Sell), Size USD, Closed PnL |

## 🔍 Methodology

1. **Data Loading & EDA** — Inspect shape, missing values, duplicates for both datasets.
2. **Timestamp Alignment** — Parse `Timestamp IST` to daily granularity; merge trades with daily sentiment on `date`.
3. **Metric Engineering** — Derive `win` flag, daily PnL per trader, trade counts, average size, long/short counts.
4. **Sentiment-Split Analysis** — Group all metrics by `classification` (Extreme Fear → Extreme Greed) to compare performance and behavior.
5. **Trader Segmentation** — Identify top/bottom performers; analyze buy vs sell breakdown per sentiment regime.
6. **Visualization** — Bar charts, pie charts, and grouped bar charts for each key question.

---

## 📈 Key Insights

1. **Average PnL varies across sentiment regimes** — Traders do not perform uniformly; certain sentiment windows produce higher average closed PnL.
2. **Win rate differs by sentiment** — The proportion of winning trades shifts between Fear and Greed days, suggesting sentiment influences trade outcome.
3. **Long/Short bias shifts with sentiment** — Buy-side dominance increases on Greed days; Fear days show more balanced or short-skewed activity.
4. **Position size reflects confidence** — Average `Size USD` is larger during Greed classifications, indicating traders size up when sentiment is positive.

---

## 🧠 Strategy Recommendations

### Rule 1 — Reduce leverage on Fear days
Traders in the bottom-performance segment show larger losses concentrated in Fear/Extreme Fear periods. Capping leverage at 2–3× during these windows can limit drawdown.

### Rule 2 — Increase trade frequency only on Greed days (for top performers)
Top-quartile traders by average PnL show higher activity and win rates specifically during Greed/Extreme Greed days. Systematic traders should consider increasing position frequency only when sentiment ≥ Greed.
